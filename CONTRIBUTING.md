# Contribuindo com o Potato-Claw

Obrigado pelo interesse em contribuir! 🥔

## Como contribuir

1. Fork o repositório
2. Crie uma branch para sua feature: `git checkout -b minha-feature`
3. Instale as dependências de dev: `make dev`
4. Faça suas alterações
5. Rode lint e testes: `make lint && make test`
6. Commit com mensagem descritiva
7. Abra um Pull Request

## Padrões

- Python 3.11+
- Linting com [ruff](https://github.com/astral-sh/ruff)
- Testes com pytest
- Mensagens de commit em português ou inglês

## Reportando bugs

Abra uma [issue](https://github.com/brunoqualhato/potato-claw/issues) com:
- Descrição do problema
- Passos para reproduzir
- Comportamento esperado vs obtido
- Versão do Python e SO

## Dúvidas

Abra uma issue com a tag `question`.
